# ✈️ War Drift

**War Drift** is a fast-paced 2D vertical shooter built with JavaScript and HTML5 Canvas. Control a fighter jet, dodge enemy attacks, and fire continuously to survive intense aerial combat.

---

## 🎮 Gameplay
Destroy enemy planes and survive as long as possible to increase your score. Simple, fast, and addictive arcade action.

---

## 🕹️ Controls

### 💻 Desktop
- **A** → Move Left  
- **D** → Move Right  
- **Left Mouse Button** → Shoot  

### 📱 Mobile  
- On-screen controls available (Left / Right / Shoot)

---

## 🚀 Features
- Smooth player movement  
- Continuous shooting mechanics  
- Score tracking system  
- Ammo management system  
- Simple & addictive gameplay
